# Core ML Algorithms

Implementations of fundamental machine learning algorithms and neural network components.

This repository documents my exploration of how machine learning models work internally by building them step-by-step.

## Implementations

### Neural Networks
- XOR Neural Network
  - Neural network architecture
  - Activation functions
  - Forward propagation
  - Loss function
  - Backpropagation
  - Training loop
  - Loss visualization using Matplotlib

## Goals

- Understand machine learning algorithms at a fundamental level
- Implement core training mechanics
- Visualize training behaviour and optimization dynamics
- Build an intuition for neural network learning

## Tech Stack

- Python
- NumPy
- Matplotlib
- Google Colab

## Motivation

This repository focuses on understanding the mechanics behind machine learning models by implementing core algorithms from first principles using NumPy, and then exploring equivalent implementations using modern ML frameworks.
